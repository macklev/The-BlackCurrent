# The BlackCurrent

For my final project, I want to create a social media site specifically for members of my house. There are 8 of us and they have actually been asking me to create something for ourselves for a while (since one of us doesn't like traditional social media).  


Users will be able to post, customize their profile, and add each other as friends to keep up with the happenings going on inside our house.
